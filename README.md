# novi

Custom bot for Nova's Discord server.
This bot is highly customized to match the server's needs or something.
Anyway, it probably won't be a use for you, but you can still get the code for free!

# Functions
The bot has functions divided into sections.
Bot's prefix is `n!`.

**Fun**:
- howfemboy <user> - calculates how femboy <user> is.
- coffee - makes you a cup of coffee.
- eightball <question> - answers your question via using God's knowledge. The communication is limited to yes/no questions tho.
- blahaj - random BLÅHAJ IMAGE!!!
**Moderation**:
- mute <user> <time> <reason (optional)> - mutes (timeouts) <user> for <time>.
- kick <user> <reason (optional)> - kicks <user>. Self-explanatory.
- ban <user> <reason (optional)> - self-explanatory.
- unban <userid> - guess.
**General**:
- ping - gets Novi's brain delay.
- about - misc info about the bot.
- repos - list project repos
- help - shows info about the commands.
